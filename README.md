## Technology stack

- React.js
- Tailwind
- Vite.js

## Running locally

1. Clone this repo:

```sh
$ git clone https://github.com/Victor-als/lofi-sounds.git
```

2. Then go to the project's folder:

```sh
cd lofi
cd LofiSounds
```

3. Install all dependencies:

```sh
npm install
```

4. Run locally:

```sh
npm run dev
```
